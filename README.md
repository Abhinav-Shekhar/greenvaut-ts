# GreenVault - TypeScript Starter (Vite + React + Tailwind)

This is a ready-to-run TypeScript Vite project with TailwindCSS, Framer Motion, and Recharts.

## Run locally

1. Install:
   ```
   npm install
   ```
2. Start dev server:
   ```
   npm run dev
   ```

Upload to StackBlitz by zipping and importing, or push to GitHub and import from GitHub.
